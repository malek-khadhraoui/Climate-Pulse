3. ğŸ§ª Data Preparation â€” scripts/data_load.R
This script loads and prepares climate datasets from the hockeystick package.

What It Does:
Loads cached .rds files from your local hockeystick cache.

Renames columns for consistency (year, co2_ppm, temp_anomaly).

Merges COâ‚‚ and temperature datasets into a tidy dataframe.

Saves the result as climate_co2_temp.csv in the data/ folder.

Sample Code:
r
library(tidyverse)

# Define cache path
cache_dir <- file.path(Sys.getenv("LOCALAPPDATA"), "R", "cache", "R", "hockeystick")

# Load datasets
co2 <- readRDS(file.path(cache_dir, "maunaloa.rds"))
temp <- readRDS(file.path(cache_dir, "gisstemp.rds"))

# Clean and align
co2 <- co2 %>% rename(co2_ppm = average) %>% select(year, month, date, co2_ppm)
temp <- temp %>% rename(temp_anomaly = `J-D`) %>% mutate(year = lubridate::year(Year)) %>% select(year, temp_anomaly)

# Merge and export
climate <- inner_join(co2, temp, by = "year")
dir.create("data", showWarnings = FALSE)
write_csv(climate, "data/climate_co2_temp.csv")
